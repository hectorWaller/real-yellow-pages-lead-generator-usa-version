The Real Yellow Pages Lead Generator scraper allows you to extract detailed business information from YellowPages.com. It simplifies the process of finding and organizing contact details across multiple industries.

### Why It Matters
- Builds verified lists of local businesses for marketing and sales.
- Automates manual data collection to save time.
- Targets specific industries and cities to improve lead relevance.
- Provides ready-to-use structured data for CRM or analytics.

## What Data This Scraper Extracts
| Field Name | Field Description |
|-------------|------------------|
| business_name | Official name of the listed business. |
| category | Business category or service type. |
| address | Complete physical business address. |
| city | City where the business operates. |
| state | State abbreviation for the business location. |
| zip_code | ZIP or postal code of the business. |
| phone_number | Primary contact phone number. |
| email | Extracted business email address (if available). |
| website | Official website URL of the business. |
| rating | Average customer rating from Yellow Pages. |
